# 🔧 MAC Processor Pro v3.1

![Version](https://img.shields.io/badge/version-3.1-blue)
![PWA](https://img.shields.io/badge/PWA-✓-success)

Procesador profesional de direcciones MAC - Cambia 1, 2 o 3 últimos dígitos.

## 🆕 **Nueva característica principal**

### **Selección de dígitos a cambiar**
Ahora puedes elegir cuántos últimos dígitos cambiar:

#### 🔸 **1 Dígito** (Modo clásico)
- Cambia solo el último dígito de la MAC
- Perfecto para modificaciones simples

#### 🔸 **2 Dígitos** (Modo avanzado)
- Cambia los últimos 2 dígitos
- Útil para configuraciones de red específicas

#### 🔸 **3 Dígitos** (Modo profesional)
- Cambia los últimos 3 dígitos
- Para casos especiales y configuraciones complejas

## 📋 **Uso**

### **Procesar MAC con N dígitos:**
1. Selecciona cantidad de dígitos a cambiar (1, 2 o 3)
2. Ingresa la dirección MAC original (12 dígitos hexadecimales)
3. Escribe los nuevos dígitos en los campos correspondientes
4. Obtén el resultado procesado (10 dígitos)
5. Cópialo en el formato que prefieras

### **Atajos rápidos:**
- `Ctrl+1` - Cambiar 1 dígito
- `Ctrl+2` - Cambiar 2 dígitos  
- `Ctrl+3` - Cambiar 3 dígitos
- `Ctrl+Enter` - Procesar MAC
- `F1` - Mostrar ayuda completa
